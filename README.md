# track-test-platform

> An electron-vue project

#### Build Setup

node 版本 <10-14>

``` bash
# install dependencies
npm install --registry=http://www.npmjs.org

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build


```
